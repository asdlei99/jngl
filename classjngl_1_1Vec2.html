Detailed Description

Two-dimensional vector.

JNGL doesn't provide any linear algebra functions but relies on the excelent Boost.QVM:

#include <boost/qvm/vec_operations.hpp>
jngl::Vec2 v(3, 4);
double length = boost::qvm::mag(v);
assert(length == 5);
